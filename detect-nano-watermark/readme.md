# Detect an AI Watermark (EyePop.ai Ability)

Automatically classify whether an image contains a common AI-generated watermark (outputs **`Watermark`** or **`No_Watermark`**).  

## Try it
- **Ability page (docs + prompt):** https://www.eyepop.ai/abilities/detect-ai-watermark

## Output
- `Watermark`
- `No_Watermark`
